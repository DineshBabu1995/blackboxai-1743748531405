# RPA Developer Resume

## Project Overview
This project is a personal portfolio website designed for a Robotic Process Automation (RPA) Developer. It showcases skills, experiences, and projects related to automation using various platforms like UiPath, Automation Anywhere, and Blue Prism. The website features a responsive design built with Tailwind CSS and includes sections such as About, Skills, Experience, Projects, and Contact.

## Installation
To get started with the project, follow these steps:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/yourusername/rpa-developer-resume.git
   cd rpa-developer-resume
   ```

2. Open the `index.html` file in your web browser to view the site.

## Usage
You can navigate the website using the top navigation bar. The sections include:

- **About**: A brief introduction highlighting the developer's qualifications.
- **Skills**: Displays technical skills with proficiency levels shown on progress bars.
- **Experience**: A list of professional experiences with details on roles and achievements.
- **Projects**: Showcases significant RPA projects undertaken by the developer.
- **Contact**: Provides a form to send messages directly to the developer.

The site allows users to toggle between light and dark modes.

## Features
- Responsive design that adapts to different screen sizes.
- Progress bars to visually represent skill levels.
- Slide-in animations using Tailwind CSS for a modern look and feel.
- Contact form with basic form validation.
- Theme toggle for dark mode support.

## Dependencies
This project uses the following external libraries:
- [Tailwind CSS](https://tailwindcss.com/) for styling.
- [Font Awesome](https://fontawesome.com/) for icons.
